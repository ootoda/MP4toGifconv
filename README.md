## はじめに

MP4動画を、SNSやブログでそのまま使える**アニメーションGIF**に変換するデスクトップアプリを作りました。

動画編集ソフトを開かなくても、
「MP4を選んで → 設定して → 変換する」だけで、
軽量で高品質なGIFを作成できます。

FFmpegを内部で使用しており、画質・サイズ・FPS・色数などを細かく調整できるのが特徴です。

---

## このアプリでできること

・ MP4動画（.mp4）を読み込み、アニメーションGIFとして出力
・ FPS（フレームレート）の指定、または元動画のFPSを維持
・ 解像度の指定／元解像度を維持／半分に縮小
・ GIFの色数を 8〜256色 から指定可能
・ 高画質／標準／軽量／超軽量 の品質プリセット
・ 無限ループGIFの生成
・ 動画の**トリミング（開始〜終了位置指定）**に対応
・ 変換進行状況をプログレスバーとログで確認可能
・ フォルダ内のMP4をまとめて変換（バッチ処理）

---

## 対応形式

### 入力（動画）

* MP4（.mp4）

### 出力（画像）

* アニメーションGIF（.gif）

※ 現在はMP4 → GIFに特化した仕様です。

---

## 画面イメージと使い方

1. 「選択」ボタンからMP4ファイル（またはフォルダ）を指定します
2. 必要に応じて、FPS・解像度・色数・プリセットを設定します
3. 「変換開始」を押すと、GIF生成が始まります
4. 変換後のGIFは自動でサブフォルダが作成されてその中に生成されます

## 技術的なしくみ（簡単に）

内部では FFmpeg を使用し、

* パレット生成（palettegen）
* パレット適用（paletteuse）

を分離して行うことで、GIF特有の色劣化を抑えています。

FPS制御・スケーリング・トリミングはすべてフィルタとして組み合わせ、
画質とファイルサイズのバランスを調整できる構成になっています。
